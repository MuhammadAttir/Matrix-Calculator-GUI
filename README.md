# Matrix Calculator (WinAPI)

A C++ Windows desktop application using WinAPI.

## Features
- 1D array operations (Add, Subtract, Multiply)
- 2D matrix operations (Add, Subtract, Multiply)
- Advanced matrix operations:
  - Determinant (2x2, 3x3)
  - Adjoint
  - Transpose

## Technologies
- C++
- WinAPI (Windows.h)
